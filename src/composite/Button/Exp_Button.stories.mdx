# Button

The Button component will render whatever is passed as in its enclosing tag as its children


<Canvas>
  <Button>Button</Button>
</Canvas>

## Usage

```jsx
import { Button } from "@qure-ui-components";

export const ExampleButton = () => {
  return <Button>Some Button</Button>;
};
```

## Variants

This button component comes with three variants

- primary
- secondary
- link

## Props

| prop     | description                                    | type                                 |
| -------- | ---------------------------------------------- | ------------------------------------ |
| children | Anything between its opening and closing tag   | ReactNode                            |
| variant  | Default is `primary`                           | `primary`, `secondary`, `link`       |
| size     | Default is `normal`                            | `small`, `normal`, `medium`, `large` |
| onClick  | A function to be called when button is clicked | function                             |
